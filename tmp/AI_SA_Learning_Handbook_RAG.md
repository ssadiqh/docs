## Foundation Concepts

### Generative AI
Creates new content from learned patterns rather than following explicit rules.

### LLM
A transformer model trained for next-token prediction that produces language, reasoning and generation capabilities.

### Tokens
The numerical units processed by models. Cost, context windows and throughput are measured in tokens.

### Parameters / Weights
Parameters are learned values. Knowledge primarily resides in weights.

### Neural Networks
Layered pattern-recognition systems that learn by adjusting weights.

### Transformers
Architecture based on attention that allows relationships between tokens to be learned efficiently.

### Attention
Determines which tokens are important to each other in context.

### Context Window
The model's working memory during a single inference call.

### Hallucination
Confident but unsupported or incorrect output generated by the model.

### Knowledge Cutoff
The latest point in time included in training data.

# Cross Encoder vs Embedding Model

Embedding Model

Question
↓
Transformer
↓
Vector

Document
↓
Transformer
↓
Vector

Vectors Compared Later

Cross Encoder

Question + Document
↓
Transformer
↓
Relevance Score

### Key Difference
Embedding models process separately.
Cross Encoders process jointly.

### Why Better?
Attention can flow between question and document.
