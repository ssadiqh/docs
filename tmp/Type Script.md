# TypeScript to AI Solution Architect Learning Path

## Foundation

| Step | Topic | Course / Resource | Link | Time | Key Learnings |
|------|--------|--------|--------|--------|--------|
| 1 | TypeScript | TypeScript Essential Training | https://www.linkedin.com/learning/typescript-essential-training-14687057 | 2-3 hrs | Types, interfaces, classes, generics, async/await, modules |
| 2 | Node.js | Node.js Essential Training | https://www.linkedin.com/learning/node-js-essential-training-14888164 | 2-4 hrs | Node runtime, npm, package.json, async programming |
| 3 | Express | Express Essentials: Build Powerful Web Apps with Node.js | https://www.linkedin.com/learning/express-essentials-build-powerful-web-apps-with-node-js | ~2 hrs | Routing, middleware, REST APIs, request lifecycle |
| 4 | NestJS | NestJS Documentation | https://docs.nestjs.com/ | 4-8 hrs | Controllers, services, dependency injection, modules, validation, authentication |

## Recommended Sequence

TypeScript → Node.js → Express → NestJS → OpenAI SDK → Embeddings → RAG → Vector Databases → Agents → LangGraph → MCP → Observability → Security & Governance
