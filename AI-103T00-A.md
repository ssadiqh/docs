# AI-103 Learning Paths \- Simple List

## Learning Path 1: Develop Generative AI Apps in Azure

**Duration:** 6 hr 52 min | **XP:** 5,700

1. **Plan and prepare to develop AI solutions on Azure** (1 hr)  
     
   - Identify AI capabilities you can implement, describe Microsoft Foundry and Foundry Tools, identify appropriate developer tools and SDKs, describe responsible AI considerations

   

2. **Select, deploy, and evaluate Microsoft Foundry models** (1 hr 3 min)  
     
   - Explore model catalog, compare models using benchmarks, deploy models to endpoints, evaluate performance using manual and automated approaches

   

3. **Develop a generative AI chat app with Microsoft Foundry** (53 min)  
     
   - Use Chat playground, choose endpoints and authentication, use Responses API and ChatCompletions API to generate responses

   

4. **Develop generative AI apps that use tools** (1 hr)  
     
   - Describe tool capabilities, use code\_interpreter, web\_search, file\_search, and function tools

   

5. **Optimize generative AI model performance with Microsoft Foundry** (2 hr 11 min)  
     
   - Apply prompt engineering techniques, ground model with RAG, fine-tune for consistent behavior, compare and combine optimization strategies

   

6. **Implement a responsible generative AI solution in Microsoft Foundry** (45 min)  
     
   - Describe responsible AI process, identify and prioritize potential harms, measure harms, mitigate harms, prepare to deploy responsibly

## Learning Path 2: Develop AI Agents on Azure

**Duration:** 9 hr 52 min | **XP:** 8,800

1. **Develop AI agents with Microsoft Foundry and Visual Studio Code** (1 hr 35 min)  
     
   - Understand AI agents, build agents in Foundry, set up VS Code, extend agent capabilities with tools, test and deploy agents

   

2. **Integrate custom tools into your agent** (53 min)  
     
   - Extend agent capabilities by integrating custom tools for specialized tasks

   

3. **Integrate MCP Tools with Azure AI Agents** (51 min)  
     
   - Connect MCP-hosted tools and integrate them seamlessly into agent workflows

   

4. **Build knowledge-enhanced AI agents with Foundry IQ** (1 hr 11 min)  
     
   - Connect agents with enterprise knowledge using RAG, improve retrieval quality, configure agent instructions for cited responses

   

5. **Integrate your agent with Microsoft 365** (1 hr 14 min)  
     
   - Publish agents to Teams and Microsoft 365 Copilot, access workplace data

   

6. **Build agent-driven workflows using Microsoft Foundry** (1 hr 4 min)  
     
   - Orchestrate AI agents and components to create intelligent applications

   

7. **Develop an AI agent with Microsoft Agent Framework** (55 min)  
     
   - Build agents using Microsoft Agent Framework SDK

   

8. **Orchestrate a multi-agent solution using the Microsoft Agent Framework** (1 hr 13 min)  
     
   - Develop agents that collaborate for multi-agent solutions

   

9. **Discover Azure AI Agents with A2A** (56 min)  
     
   - Enable agent discovery, direct communication, and coordinated task execution across remote agents

## Learning Path 3: Develop Natural Language Solutions in Azure

**Duration:** 5 hr 46 min | **XP:** 5,700

1. **Analyze text with Azure Language in Foundry Tools** (46 min)  
     
   - Extract semantic information from text, detect language, extract entities, extract PII

   

2. **Develop a text analysis agent with the Azure Language MCP server** (52 min)  
     
   - Build agents for language detection, entity recognition, PII redaction

   

3. **Develop a speech-capable generative AI application** (43 min)  
     
   - Build applications with speech transcription and synthesis

   

4. **Create speech-enabled apps with Azure Speech in Microsoft Foundry Tools** (53 min)  
     
   - Use speech-to-text and text-to-speech APIs

   

5. **Develop a speech agent with the Azure Speech MCP server** (52 min)  
     
   - Build agents performing speech-to-text and text-to-speech tasks

   

6. **Develop an Azure Speech Voice Live Agent in Microsoft Foundry** (52 min)  
     
   - Create Voice Live agents for conversational AI

   

7. **Translate text and speech with Microsoft Foundry Tools** (48 min)  
     
   - Create intelligent translation services for text and speech

## Learning Path 4: Extract Insights from Visual Data on Azure

**Duration:** 7 hr 6 min | **XP:** 6,500

1. **Develop a vision-enabled generative AI application** (43 min)  
     
   - Build multimodal chat apps that interpret images and respond to visual prompts

   

2. **Generate images with AI** (33 min)  
     
   - Create original images from natural language prompts

   

3. **Generate videos with Microsoft Foundry** (48 min)  
     
   - Generate videos from text prompts with Sora 2

   

4. **Analyze images with Content Understanding** (43 min)  
     
   - Use Azure Content Understanding for image analysis

   

5. **Create a multimodal analysis solution with Azure Content Understanding** (1 hr)  
     
   - Perform multimodal content analysis and information extraction

   

6. **Create an Azure Content Understanding client application** (1 hr)  
     
   - Use Content Understanding API for multimodal analysis

   

7. **Extract data with Azure Document Intelligence** (1 hr 11 min)  
     
   - Use OCR and deep learning to extract text, key-value pairs, tables from documents

   

8. **Create a knowledge mining solution with Azure AI Search** (1 hr 8 min)  
     
   - Implement knowledge mining to extract, enrich, and make data searchable
